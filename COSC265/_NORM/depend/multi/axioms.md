
# MVD AXIOMS



Complementation: 
If X  ->->  Y, then X  ->->  R - Y


Augmentation:
If X  ->->  Y and Z _subset-of_  W, then XW  ->->  YZ


Transitivity: 
If X  ->->  Y and Y  ->-> Z, then X  ->->  Z - Y


Replication:
If X  --->  Y, then X  ->->  Y

Coalescence: If X  ->-> Y and there exists  W such that
W  _intersection_  Y =  _empty_, 
W  --->  Z, and 
Z  _subset-of_  Y, then
    X  --->  Z
