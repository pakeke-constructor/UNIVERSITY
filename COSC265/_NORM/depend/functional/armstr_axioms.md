
# Armstrongs axioms



# Reflexivity
    If Y is a subset of X,
       X  --->  Y.

EXAMPLE:
{name} is subset of {id, name}.
Therefore,   {id, name}  --->  {name}







# Augmentation
    If X  --->  Y,
        XZ  --->   YZ

EXAMPLE:
{id}  --->  {dob}
therefore,   {id, name}  --->  {dob, name}






# Transitivity
    If   X ---> Y    and    Y ---> Z,
        X ---> Z.

EXAMPLE:

{id}  --->  {name, last_name, dob}
{name, last_name, dob}  --->  {course}

therefore,   {id}  --->  {course}
