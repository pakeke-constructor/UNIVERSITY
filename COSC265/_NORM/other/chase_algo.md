

# CHASE ALGORITHM
We can also check using the chase algorithm.

Example:
We have a relation 
    
    R1(A1, A2, ... Ak-1, Ak)

that gets decomposed into  `n` relations:

    R1, R2, ... Rn.

To check if the decomposition was lossy/lossless,
write out a K x N table.
